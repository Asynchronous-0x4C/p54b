図形の描画

図形の描画

四角を描く

rect(x, y, w, h);

  • x：x座標
  • y：y座標
  • w：横幅
  • h：高さ

円を描く

ellipse(x,y,w,h);

ellipseの場合、

  • w:横方向の直経
  • h:縦方向の直径 を表す。

三角形を描く

triangle(x1,y1,x2,y2,x3,y3);

サンプル

size(400, 400);
triangle(200, 20, 40, 300, 300, 350);

基本はこの三つです
